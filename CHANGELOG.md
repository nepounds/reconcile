# Changelog

## 0.1.0 - Portfolio release candidate

- Added event-sourced double-entry ledger core.
- Added SQLite schema and append-only ledger event store.
- Added account opening, journal posting, and reversal workflows.
- Added rebuildable projections and accounting reports.
- Added direct-method cash flow with AR/AP operating-cash refinement.
- Added bank CSV import, normalization, and duplicate detection.
- Added exact, fuzzy, and split reconciliation.
- Added rule-based categorization, corrections, and local classifier.
- Added Streamlit read-only dashboard.
- Added GitHub Actions CI.
- Added fake demo data and sample output CSVs.
